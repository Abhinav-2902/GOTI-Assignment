# GOTI-Assignment
This assignment focuses on building a responsive, production‑quality homepage for Kilangi Jewellery, based on a given desktop design and a mobile reference (PDF).

Tech Stack:
HTML5 – semantic structure
CSS3 – Flexbox, Grid, responsive media queries
Vanilla JavaScript – lightweight, framework‑free
No external UI frameworks were used.

Project Structure:
project/
│── index.html
│── style.css
│── assets/
│ ├── images/
│ └── icons/
└── README.md

How to Run:
1.Clone or download the repository
2.Open index.html in any modern browser
3.Resize the browser or use dev tools to view mobile & tablet layouts
4.No build steps or dependencies required.
